# AI Software Engineering Team

An autonomous multi-agent system that plans, designs, generates, tests, reviews, and iteratively improves software projects.

## Features

- Manager Agent
- Planner Agent
- Architect Agent
- Coder Agent
- Tester Agent
- Reviewer Agent
- LangGraph Workflow
- PostgreSQL Memory

## Tech Stack

- Python 3.12
- LangGraph
- PostgreSQL
- OpenAI API
- Pytest
- Docker
